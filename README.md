# SocialNet

SocialNet-Web is a modern LinkedIn-inspired social networking UI built using HTML and CSS.  
The project focuses on recreating a clean and responsive social media interface with modern frontend design and layout techniques.

---

## 🌐 Live Demo

https://social-nest-nine.vercel.app/

---

## 🚀 Features

- Responsive LinkedIn-style layout
- Interactive navigation bar
- Profile sidebar section
- Feed post creation UI
- Social media post cards
- LinkedIn News section
- Puzzle/Games widget
- Floating messaging bar
- Hover effects and modern UI styling
- Clean and structured design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
